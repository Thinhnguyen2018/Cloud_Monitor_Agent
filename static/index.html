Tính năng
Quan sát
  • Liệt kê VM / Volume / SG / Network / FIP
  • Health check — phát hiện SHUTOFF/ERROR
  • Orphan: volume chưa gắn, FIP idle
  • Security audit — rule mở 0.0.0.0/0
  • Topology VM → SG → VPC
  • Whoami — quyền service account
Actions
  • Start / stop / reboot VM (có xác nhận)
  • Tạo / xóa VM mới
  • Resize VM (đổi flavor)
  • Tạo snapshot VM
  • Đổi tên VM / Volume
  • Tạo / xóa Volume
  • Attach / detach Volume
  • Gắn / gỡ Security Group
  • Thêm / xóa SG Rule
  • Associate / disassociate Floating IP
Lịch hẹn
  • Hẹn giờ start VM lúc 6:00 sáng
  • Hẹn giờ stop VM lúc 22:00
  • Xem danh sách lịch đã đặt
  • Hủy lịch bất cứ lúc nào
Multi-tenant
  • Lưu credentials nhiều KH vĩnh viễn
  • Switch KH không cần nhập lại key
  • Chat history cách ly hoàn toàn
  • Giao diện end-user riêng: /customer
